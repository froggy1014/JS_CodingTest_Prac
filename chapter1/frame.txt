 ### ❓ Question
 
 <pre>알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 <br>반복되는 경우 반복되는 문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로<br>
  문자열을 압축하는 프로그램을 작성하시 오. 단 반복횟수가 1인 경우 생략합니다.
 </pre>


 <br>
 
 ### ‼️ Solution
 
 ```javascript
function solution(str)
{
  let answer = "";
  let letter ;
  let cnt = 1;
  for (x of str ){
    if (letter != x){
      if ( cnt > 1 ){
        answer += String(cnt)
        cnt = 1;
      }
      letter = x
      answer += x;
    }else cnt++;
  }
  return answer;
}
const str = "KKHSSSSSSSE"
console.log(solution(str))
 ```
